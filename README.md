# predicting-air-quality

Predicts PM2.5 air quality levels in New York City using weather and time-based features with machine learning models.

## Requirements
- Python 3.11+
- Install dependencies:

pip install -r requirements.txt

## Project Structure

├── data/
│   ├── raw_data/          # Raw PM2.5 CSVs and wind data
│   └── processed_data/    # Generated by running notebooks
├── notebooks/
│   ├── combine_data.ipynb                    # Combines raw PM2.5 CSVs
│   ├── wind_pressure_preprocessor.ipynb      # Processes wind data
│   └── NYC_data_processor.ipynb              # Feature engineering, models, plots
├── requirements.txt
└── README.md

## Usage
Run notebooks in order:
1. Delete any existing files in data/processed_data/ before running the notebooks
2. combine_data.ipynb - Combines 12 raw CSVs and outputs NYC_combined_data.csv to data/processed_data/
3. wind_pressure_preprocessor.ipynb - Processes raw wind data and outputs NYC_wind_pressure.csv to data/processed_data/
4. NYC_data_processor.ipynb - Reads processed CSVs, trains and evaluates all models, and generates plots

## Features
- Temperature, relative humidity, wind speed, wind direction, atmospheric pressure
- Time-based features: hour, day of week, month, rush hour, weekend flags

## Models & Results
| Model             | R²   | RMSE |
|-------------------|------|------|
| Linear Regression | 0.16 | 6.67 |
| XGBoost           | 0.69 | 4.08 |
| Random Forest     | 0.75 | 3.66 |

## Data Sources
- PM2.5/temperature/humidity data: OpenAQ (https://openaq.org/)
- Wind/pressure data: ASOS Network (JRB station) (https://mesonet.agron.iastate.edu/request/download.phtml?network=NY_ASOS)
